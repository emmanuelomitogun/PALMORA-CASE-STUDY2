## 🚀 Project Overview

A comprehensive gender equity and compensation analysis for Palmoria Group, a Nigerian manufacturing company. This analysis revealed **₦2.9M in compensation gaps** and provided strategic recommendations for achieving workplace equity across 946 employees in three regional offices.

**Key Impact**: Transformed a reputation crisis into strategic organizational development, ensuring fair compensation and sustainable growth.
